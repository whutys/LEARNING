# Windows下使用Hexo

Hexo官方help地址 https://hexo.io/docs/

# 1.安装git
http://git-scm.com/

# 2.安装node.js
http://nodejs.org/
检验 node -v

# 3.安装Hexo

# 4.打开git，输入npm install -g hexo-cli
国内加上  --registry=https://registry.npm.taobao.org  重要
检验 hexo -v

# 5.创建博客文件夹，如D:\\blog

# 6.在文件夹内右键打开Git，或cd到文件夹内

# 7.hexo init
初始化

# 8.npm install
安装依赖
国内加上  --registry=https://registry.npm.taobao.org  重要

# 9.hexo g
generate生成博客静态资源文件，public文件夹下

# 10.hexo s
server服务启动，默认端口4000
